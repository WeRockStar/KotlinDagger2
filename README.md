### KotlinDagger2 
=====
A simple build android app with `Kotlin` office language for android.

#### Libraries
- Dagger2
- RxJava2
- RxAndroid2
- Retrofit
- OkHttp
- Support libraries
- Junit - Testing 
- Mockito - Testing 
- Robolectric - Testing

#### Support testing
- Unit Test - Junit
- UI Test - Espresso
- E2E Test - Calabash android
